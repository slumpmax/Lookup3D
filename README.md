# Loopup3D
- Preview &amp; Fix 3D object files
- Source code in Delphi Community Edition v11.3

## Format supports
- 3DS
- OBJ
- RAW
- DAE (Not available now)

## Features
- Select texture for any object or material
- Set view
- Resize object to visible area
- Export to OBJ
- Etc
